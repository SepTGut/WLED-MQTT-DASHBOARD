# Next Dashboard (Simple)

This is a simple Next.js + Tailwind version of your MQTT dashboard, keeping the previous layout style.

## Run

1. `cd next-dashboard`
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:3000`

## Notes

- Uses app router (`app/`).
- Uses Tailwind CSS.
- Loads Paho MQTT from CDN at runtime.
- Intended as a clean starting point for gradual migration from the current static app.
